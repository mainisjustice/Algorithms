	В программе реализован классический алгорим Флойда-Уорелла.
	Чтобы скомпелировать программу необходимо выполнить файл build.sh.
	После компиляции создается папка bin, а в ней исполняемый файл floyd_warshall.out.

	На вход программы подается сперва количество вершин графа (kol), после этого матрица смежности графа размерностью kolxkol. Значение "0" в матрице означает, что пути ребра соединяющего вершины нет. Рассматриваются графы без циклов в вершинах (это регулируется один условием в коде алгоритма). Пример входных данных (3 вершины в графе):
	3
	0 3 1
	3 0 1
	1 1 0
	На выходе программа выдает двумерный массив длин минимальных путей между вершинами графа. Пример выходных данных:
	0 2 1
	2 0 1
	1 1 0
	Алгоритм Ф-У запрограммирован в функции minFL, аргументы которой - m (указатель на двумерный массив матрицы смежности графа) и kl (количество вершин графа).
	В программе закоментированы все текстовые поясняющие выводы, чтобы они не мешали автоматическому тестированию.
